# Garment Authoring

This directory holds curated garment source assets for the mannequin-first fitting pipeline.

Expected structure:

- `blender/`
  - editable garment source files grouped by category
- `exports/raw/`
  - unoptimized GLB exports
- `exports/optimized/`
  - validated runtime candidates

Every promoted garment must match the shared skeleton profile and the fitting contract in `docs/garment-fitting-contract.md`.
