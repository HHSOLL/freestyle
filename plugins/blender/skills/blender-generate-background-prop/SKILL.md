# Blender Generate Background Prop

Use this skill for practical environment filler assets.

## Workflow

1. Confirm the prop type, target style, and output path.
2. Use `blender_run_python` to build the prop from primitives, curves, and light modifier usage.
3. Keep the prop readable at medium distance:
   - simple silhouette
   - limited material slots
   - modest detail density
4. Save a `.blend` source and export a `.glb` if requested.

## Good Fits

- storage bins, cardboard boxes, baskets, plant pots
- stools, hangers, shelf decor, display risers
- background bags, folded cloth stacks, neutral containers
